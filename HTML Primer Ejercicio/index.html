Ejercicio 1: HTML.

  • body: Contiene el cuerpo principal de la página, es decir, todo el contenido visible en el navegador.
  • head: Contiene información sobre el documento, como el título, la meta-descripción y los enlaces a hojas de estilo.
  • p: Define un párrafo de texto.
  • a: Define un enlace hipertexto a otra página web o a una sección dentro de la misma página.
  • img: Inserta una imagen en la página.

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).
